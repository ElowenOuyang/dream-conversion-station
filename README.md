# Dream Conversion Station

A web-based speculative medium for transforming dreams.
AMCC 6500E Final Project.

## Concept

Dreams cannot be transmitted. They can only be converted.

This interactive web artwork takes personal dream fragments and processes them through interpretation, prompt translation, receiver filtering, distortion, and AI generation — producing a transformed dream object: an image, a poetic text, and system metadata.

## Live Demo

Visit: [https://elowenouyang.github.io/dream-conversion-station](https://elowenouyang.github.io/dream-conversion-station)

## How It Works

1. **Submit** dream fragments (what you saw, felt, heard)
2. **Choose** a receiver mode (Anxious, Nostalgic, Detached, Romantic, Unknown)
3. **Witness** the conversion process
4. **Receive** a converted dream letter with AI-generated abstract image and metadata

The original dream input is never displayed again — reflecting the irreversible nature of memory.

## Tech

- Pure HTML/CSS/JavaScript
- p5.js for abstract generative art
- No backend, no data storage
- Deployed via GitHub Pages
